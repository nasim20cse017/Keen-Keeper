# 🍃 KeenKeeper

**KeenKeeper** is a personal relationship management tool designed to help you nurture and maintain meaningful connections with friends, family, and colleagues.

### 🚀 Description
Never lose track of the people who matter. KeenKeeper allows you to set relationship goals, track your last interactions, and view analytics on how you communicate.

### 🛠️ Technologies Used
- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS & DaisyUI
- **Icons:** React Icons (Hi2, Io5)
- **Visuals:** Recharts (Analytics)
- **Feedback:** React Toastify
- **Data:** React Context API (Shared Global State)

### ✨ Key Features
1. **Interaction Logging:** Record calls, texts, and video chats with a single click from a friend's profile.
2. **Smart Timeline:** A comprehensive history of all your social interactions with advanced search, filtering, and sorting capabilities.
3. **Friendship Analytics:** Visual representation of your communication habits through interactive pie charts to see which interaction methods you use most.
